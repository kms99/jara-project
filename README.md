# Jara-Project
React hook을 사용하여 만든 쇼핑몰 페이지입니다. (컨셉 : Jara)

## 프로젝트 소개
총 3개의 페이지를 생성하며 react router 기술을 활용하여 페이지 이동이 가능합니다.

## 주요 항목
반응형 웹 디자인 (bootstrap)
React
React hook - React Router(react-router-dom), useState, useEffect
json server (create local data)
